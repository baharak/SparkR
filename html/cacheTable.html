cacheTable {SparkR}R Documentation

Cache Table

Description

Caches the specified table in-memory.

Usage

cacheTable(sqlContext, tableName)

Arguments

sqlContext

SQLContext to use

tableName

The name of the table being cached

Value

DataFrame

Examples
