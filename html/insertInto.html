insertInto

Description

Insert the contents of a DataFrame into a table registered in the current SQL Context.

Usage

## S4 method for signature 'DataFrame,character'
insertInto(x, tableName, overwrite = FALSE)

insertInto(x, tableName, ...)

Arguments

x

A SparkSQL DataFrame

tableName

A character vector containing the name of the table

overwrite

A logical argument indicating whether or not to overwrite the existing rows in the table.

Examples

##D sqlContext <- sparkRSQL.init(sc)

##D df2 <- read.df(sqlContext, path2, "parquet")
